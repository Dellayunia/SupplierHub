### Penjelasan Navigasi

| Layer | Jenis | Screen |
|---|---|---|
| **Root** | Conditional | Splash вҶ’ Auth Check |
| **Auth Stack** | Stack Navigator | Login вҶ’ Register вҶ’ Profile Setup |
| **Main App** | Bottom Tab Navigator | Explore В· Applications В· Profile |
| **Nested Stacks** | Push / Modal | Partner Detail вҶ’ Apply Form |

---

## Tech Stack

| Teknologi | Fungsi | Alasan Pemilihan |
|---|---|---|
| **React Native + Expo SDK 52** | Framework mobile | Cross-platform, developer experience terbaik, ecosystem besar |
| **Expo Router v4** | Navigasi file-based | Otomatis routing berdasarkan folder, mendukung tabs & stacks |
| **Zustand** | State management | Ringan (~2KB), mudah dipakai, tidak perlu boilerplate |
| **Firebase Auth** | Autentikasi | Email/password login siap pakai, gratis untuk skala MVP |
| **Cloud Firestore** | Database | Realtime sync, skema fleksibel, Firebase SDK terintegrasi |
| **Google Places API** | Data mitra | Rating, review, lokasi, foto вҖ” semua dari satu API |
| **react-native-maps** | Peta interaktif | Library paling stabil untuk Google Maps di React Native |
| **expo-location** | GPS pengguna | Mendapatkan koordinat supplier untuk kalkulasi jarak |

- `npx expo start` вҖ” pastikan app bisa launch tanpa error

5. Pastikan **semua status badge** tampil dengan warna yang benar
